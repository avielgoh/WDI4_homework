# Quiz - Dice

Write a program to generate a standard dice roll.

```
Dice.roll
# => 6
```
It should give you a different result each time.
```
Dice.roll
# => 5
Dice.roll
# => 2
```

When you pass in a number, it rolls the dice that many times, and returns you the array of rolled dice
```
Dice.roll(3)
# => [2,5,1]
```
